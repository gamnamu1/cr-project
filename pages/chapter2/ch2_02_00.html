2-0. 기본 유닛(Unit) 구성

100개의 유닛·허브가 동시다발적으로 생겨나 각자의 플랫폼을 실험하고 또 전략적으로 연대할 수 있다면···

저널리즘 영역

⦿ 미디어 전문가
  • 구성 : 언론 문제에 공감하는 모든 시민 + 비판적 언론인 (언론학자 + 미디어 비평 전문가 + 저널리즘 실무 전문가).
  • 역할 : 기사 품질 평가 기준 개발, 나쁜 저널리즘 사례 분석, 평가 방법론 설계
  • 이점 : 학술 이론과 현장 경험을 결합한 균형 잡힌 품질 지표 개발 가능

기술 영역

⦿ AI 전문가
  • 구성 : AI/ML 엔지니어 + 자연어처리 전문가 + 프롬프트 엔지니어 + 데이터 사이언티스트
  • 역할 : 품질 평가 알고리즘 개발, 데이터 처리, 모델 훈련 및 최적화
  • 이점: 언어모델 기반 평가 시스템 구축에 필요한 모든 기술적 역량 보유
플랫폼 개발자
  • 구성 : 시스템 아키텍트 + 웹 개발자(프런트/백엔드) + 정보보안 전문가 + UI/UX 디자이너
  • 역할 : 웹 플랫폼 설계 및 구축, 사용자 경험 최적화, 보안 체계 수립
  • 이점 : 기술적 설계부터 사용자 경험까지 아우르는 통합적 플랫폼 구축 능력

관리 영역

⦿ 법률 자문가
  • 구성 : 법률가 + 정보인권 관리자 + 저작권 전문가 + 법적 리스크 분석가
  • 역할 : 프로젝트의 법적 위험 관리, 개인정보보호 체계 구축, 저작권 침해 예방, 면책 조항 설계, 이의제기 절차 수립, 법적 운영 구조 자문
  • 이점 : 법적 안정성 확보, 언론사 대응 전략 수립, 지적재산권 보호, 사용자 데이터 보호 체계 구축으로 프로젝트의 법적 지속가능성 강화
⦿ 경영 자문가
  • 구성 : 경영 및 조직 운영 전문가 + 재정 전문가 + 비영리 조직 운영 경험자 + 프로젝트 관리 전문가
  • 역할 : 프로젝트 전략 수립, 예산 계획 및 관리, 지속가능한 운영 모델 개발, 인력 구성 및 조직 설계, 장기 로드맵 구축, 위기 관리 계획 수립
  • 이점 : 자원의 효율적 배분, 비영리 모델의 재정적 지속가능성 확보, 조직 거버넌스 체계 수립, 체계적인 성장 경로 설계로 프로젝트의 장기 생존력 강화
⦿ 홍보 전문가
  • 구성 : 홍보 전문가 + 마케터 + 캠페이너 + 커뮤니티 매니저 + 미디어 교육 전문가
  • 역할 : 사용자 참여 유도, 미디어 리터러시 교육, 사회적 영향력 확대
  • 이점 : 시민 참여와 교육을 통한 플랫폼의 사회적 확산력 강화

각 전문가는 여러 영역의 기술을 통합하여 프로젝트에 기여하며, 필요에 따라 개인이 여러 역할을 유연하게 수행할 수 있음.
