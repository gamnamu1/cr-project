10-2. 결과 전달 및 시각화 시스템

10-2-1. 계층적 평가 결과 구조

최종 평가 리포트 핵심 구성요소

평가 리포트는 단순한 점수 나열이 아닌, 체계적이고 교육적으로 구성할 필요. 다음 요소들이 균형 있게 포함되어야 한다:

  • 기본 메타 정보
    • 기사 제목 및 원문 링크
    • 언론사명/기자명/발행일자
    • 기사 유형(스트레이트/해설/인터뷰 등)
    • 글 분량 및 구조(단어 수, 단락 수, 구조적 특성)
  • 종합 평가 정보
    • 종합 품질 점수(0점 만점, 문제 발견시 마다 감점, -점수 부여)
    • 품질 등급(신호등 표시 등 다양한 방식)
    • 한 줄 평가 요약
    • 핵심 강점 및 약점 요약(3줄 이내)
  • 차원별 세부 평가
    • 8개 평가 차원별 점수 및 핵심 피드백
    • 문제 패턴 발견 빈도 및 심각도
    • 발견된 문제와 관련 언론 규범/윤리 지침 연결
  • 교육적 요소
    • 미디어 리터러시 관련 팁
    • 각 평가 기준에 대한 간략한 설명
    • 관련 개념 용어집(glossary)
  • 비교 분석 정보
    • 동일 주제 타 매체 기사와의 품질 비교
    • 해당 언론사의 시간 경과에 따른 품질 변화 추이
    • 유사 주제 기사의 모범 사례 참조

10-2-2. 시각화 전략

정보 계층화 구조

기사 품질 평가 결과를 사용자에게 효과적으로 전달하기 위한 다층적 시각화 전략을 적용:

  • Tier 1 (즉시 표시) : 종합 품질 점수와 핵심 1-3개 문제점
  • Tier 2 (탭/섹션으로 구분) : 8개 평가 차원별 점수와 주요 이슈
  • Tier 3 (클릭/확장 시 표시) : 30개 세부 패턴 분석 결과
  • 인지 부하 이론(Cognitive Load Theory)에 기반한 접근으로, 사용자가 한 번에 처리할 수 있는 정보량을 고려한 설계

소셜 미디어 최적화 시각화 요소

  • '신뢰도 배지(Credibility Badge)' : A+부터 F까지 학점 형태로 표시
  • 언론사 로고, 기사 제목 일부, 평가 날짜를 포함해 맥락 정보 제공
  • 배지 색상은 점수에 따라 차별화(A+/A: 녹색, B+/B: 파란색, C+/C: 노란색, D+/D: 주황색, F: 빨간색)
  • '문제 패턴 아이콘' : 주요 문제 패턴을 직관적인 아이콘으로 표시
  • '비교 맥락 그래프' : 해당 기사의 평가 점수를 동일 언론사 평균, 해당 주제 기사들의 평균과 비교하는 간단한 바 그래프
  • QR 코드나 단축 URL을 카드에 포함시켜 전체 분석 리포트 페이지로 쉽게 이동 가능

설명 가능한 AI(XAI) 요소 통합

  • 특정 평가의 근거를 간결하게 제시
    "이 기사는 5개의 무주체 피동형 표현을 포함하고 있어 투명성 점수가 낮습니다"
  • 복잡한 알고리즘적 판단을 자연어로 변환하여 제시
  • 신뢰도 지표 함께 표시 (예: "이 분석의 확신도: 85%")
  • 레이더 차트로 8개 평가 차원 점수 시각화
  • 문제적 표현에 색상 코드 하이라이트 적용 및 마우스 오버 시 툴팁으로 문제점 설명

10-2-3. 사용자 친화적 개선 제안

"Before/After" 방식

  • 문제 표현과 개선된 표현 병행 제시
    예: "최근 관측된다" → "연구팀이 최근 관측했다"
  • 맥락 중심 하이라이트: 문제 표현을 포함한 문장/문단 단위로 표시
  • 문제 유형별 시각적 코드화: 유형별 다른 색상과 아이콘 사용

단계적 설명 구조

  1. 무엇이 문제인지 명시
  1. 왜 문제인지 저널리즘 원칙 연결 설명
  1. 어떻게 개선할 수 있는지 대안 제시
  1. 이러한 개선이 왜 중요한지 사회적 영향 설명

계층적 정보 설계 기반 시각화

평가 결과의 효과적인 시각화를 위해 계층적 정보 설계(Layered Information Architecture)를 적용한다:

  • 1단계(상단) : 직관적인 종합 점수를 시각화. 100점 만점의 큰 숫자와 함께 A+부터 F까지의 등급을 컬러 코딩(녹색-노란색-빨간색)으로 표시하여 즉각적인 품질 인식이 가능하게 함(선택 사항).
  • 2단계(중단) : 8개 평가 차원(진실성, 투명성 등)을 레이더 차트(Spider Chart)로 표현합니다. 각 차원에 마우스를 올리면 간단한 설명과 점수가 표시됨.
  • 3단계(하단) : 원문 기사에 인터랙티브 하이라이트를 적용. 문제적 표현(무주체 피동형, 익명 취재원 등)에 색상 코딩된 밑줄을 그어 시각적으로 표시하고, 사용자가 클릭하면 다음 정보가 포함된 툴팁이 나타난다(저작권 문제로 기사원문을 그대로 시각화하기 어려울 수도 있음):
    • 문제 유형(예: "무주체 피동형 표현")
    • 영향받는 평가 차원(예: "투명성 -20점, 진실성 -15점")
    • 개선 제안(예: "출처를 명확히 명시하세요")
    • 추가 학습 자료 링크
